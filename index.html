Seamline Marker — quick start

WORKFLOW

  1. New Order — a 5-step Easy Order form: add one or more models, set size quantities per model, untick pieces to exclude, set each piece's lay direction and fabric category (any code — A/B/C, 1/2/3, BD/Gst/FR…), choose fabric width/buffer/rotation rule, review the summary. Pieces sharing a fabric code cut from the same fabric; chips above the tray filter the tray and scope Auto-Nest to one fabric at a time.
  2. Place pieces — click a tray piece, then click the fabric; or hit ⚡ Auto-Nest to pack everything automatically.
  3. Tabs & comparison — use Open in new tab… for another editable marker, or Import comparison… to overlay a read-only marker in the active field. The Layers dock controls visibility, opacity, overlay / side-by-side / difference view, and independent X/Y position. Each tab keeps its own marker, camera, history, Cut Plan, and layer setup; a dot on the tab means unsaved changes.
  4. Refine — select a piece and use the right dock's Piece tab to rotate, flip, butt/slide, centre it in the width, or return it; or drag it on the fabric. Compact tightens the whole marker. The Options tab gathers grid/snap, rotation rule, buffer, width, end allowance and the theme. The ribbon tabs above the canvas (File · Piece · Marker · Fabric · View) mirror the classic Easy Marking layout — measure tools, marker flips, fit piece, zoom window, thumbnail and restore points live there.
  5. Output — when the status chip reads MADE, export the plot as SVG (1 unit = 1 cm) or PNG, copy it 1:1 to the clipboard, or save the marker as JSON. On the Advanced tab, sequence the cut (Auto sequence = banded serpentine sweep; Pick order = click pieces in order; Knife path shows plunge points, direction and dry-haul) and Generate Cut Data as HPGL (.plt) for the cutter or DXF R12 for CAM import — entity order is the cut order.
  6. Fabric tab — for spreading: Mark defect zone (drag over a flaw; pieces keep clear) and Add splice / Auto splice (roll joins no piece may straddle). Guides added over placed pieces flag them in red and block cut-data export until you move them clear (or Nudge offenders).
  7. Cut Plan (P) — for bulk production: enter demand per colour and size — or ⇪ Import buy a customer cut-order (paste from Excel or load a CSV; it maps the columns and fills the grid) — plus ply limits, hit Calculate markers, and Seamline splits it into a marker set with spreads (colour × plies). Auto-nest all markers draws every layout and fills in actual lengths; Mini-markers PDF downloads one page per marker with the drawing and all details. Open any plan row as a marker order, export cut tickets as CSV.

RULES

  • Pieces can never overlap — an invalid drop bounces back.
  • Each piece carries a lay direction: body pieces are two-way (0/180°), small parts turn 90°, bindings are free. The Rotation rule selector (Fabric tab) can relax this for non-directional fabric. Note: in Gerber terms this is piece rotation — Gerber's "Layrules" (saved placement templates) are a planned feature.
  • Buffer keeps a minimum gap between pieces for the cutter.
  • Status: MADE = all placed · PARTIAL = some · UNMADE = none.
