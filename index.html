Seamline Marker — quick start

WORKFLOW

  1. New Order — a 5-step Easy Order form: add one or more models, set size quantities per model, untick pieces to exclude, choose fabric width/buffer/lay rule, review the summary. All models nest together in one combined marker.
  2. Place pieces — click a tray piece, then click the fabric; or hit ⚡ Auto-Nest to pack everything automatically.
  3. Refine — drag pieces, rotate/flip, Butt slides a piece left until it touches its neighbour, Compact tightens the whole marker.
  4. Output — when the status chip reads MADE, export the plot as SVG (1 unit = 1 cm) or PNG, or save the marker as JSON.
  5. Cut Plan (P) — for bulk production: enter demand per colour and size plus ply limits, hit Calculate markers, and Seamline splits it into a marker set with spreads (colour × plies). Auto-nest all markers draws every layout and fills in actual lengths; Mini-markers PDF downloads one page per marker with the drawing and all details. Open any plan row as a marker order, export cut tickets as CSV.

RULES

  • Pieces can never overlap — an invalid drop bounces back.
  • Each piece carries a lay rule: body pieces are two-way (0/180°), small parts turn 90°, bindings are free. The Lay selector can relax this for non-directional fabric.
  • Buffer keeps a minimum gap between pieces for the cutter.
  • Status: MADE = all placed · PARTIAL = some · UNMADE = none.
